# Bigcartel::Sass

A standard set of variables, mixins, animations, and placeholder selectors to use across all of Big Cartel's front-end development.

## Installation

Add this line to your application's Gemfile:

    gem 'bigcartel-sass'

And then execute:

    $ bundle
